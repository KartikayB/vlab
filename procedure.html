Flow Control

Procedure

  1. The aim is to introduce the students with the working of the three mentioned flow control protocols.
  2. A choice as to which protocol is to be simulated is given.
  3. Select the choice of protocol out of the given three choices.
  4. Once a protocol is selected, the experiment begins and an array is displayed.
  5. The user is instructed through the pop ups as to what selection is to be made in the given array buttons which ask about the frames to be sent.
  6. If the user selects the correct frames, the simulation is further played.
  7. If wrong frames are selected, the user is warned and allowed to retry.
  8. Further, in Go-Back-N and Selective Repeat protocols, a choice for the window size is also provided.
  9. The user is given a choice of selecting the window size from the drop down list, consisting of options 3,4 and 5.
  10. The user at any time can switch to any of the “window sizes” and continue to learn more about the same.
